# Knowledge Base

The KB contains reusable engineering knowledge only.

## Scope

- `10-principles/` contains stable cross-project reasoning rules.
- `20-standards/10-core/` contains core technical standards.
- `20-standards/20-language/` contains language-specific standards.
- `20-standards/30-framework/` contains framework-specific standards.

## Precedence

Within the KB, higher-precedence knowledge appears earlier:

1. `10-principles`
2. `20-standards/10-core`
3. `20-standards/20-language`
4. `20-standards/30-framework`

Lower-precedence KB layers may refine earlier layers, but may not contradict or
weaken them.
